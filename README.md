# SkillGap AI - Adaptive Skill Intelligence Platform

A premium, futuristic frontend-only web application for personalized skill assessment and AI-guided learning.

## 🎨 Features

- **Domain Selection**: Immersive grid of 15+ learning domains
- **Technology Selection**: Dynamic tech stack selection
- **Adaptive MCQ Test**: AI-powered assessment with difficulty scaling
- **AI Analysis**: Detailed skill gap analysis with strengths and weaknesses
- **Auto Roadmap Generation**: 28-day personalized learning timeline
- **AI Mentor Chat**: Interactive learning assistant

## 🚀 Tech Stack

- React 18
- Tailwind CSS
- Framer Motion
- Lucide React

## 🛠️ Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone <repo-url>
cd skillgap-ai

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:5173`

## 📦 Build for Production

```bash
npm run build
```
